# Ense-aTech
Aplicación basada en html, version 1.2
# Cómo usar ADA enTech

1. Extrae el archivo ZIP
2. Abre la carpeta `ADA-enTech`
3. Ejecuta `index.html` en un navegador
4. Para todas las funciones, usa un servidor local

## Servidor local rápido:
- Opción A: Instala "Live Server" en VS Code
- Opción B: Ejecuta: `python -m http.server 8000`
- Opción C: Usa XAMPP o WAMP
